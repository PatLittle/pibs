# Transparence

Le gouvernement du Canada s'est engagé à ouvrir le gouvernement, ce qui se fait selon trois axes: les données ouvertes, les informations ouvertes et le dialogue ouvert.

À l'appui de ces principes du [gouvernement ouvert](https://open.canada.ca/fr), nous fournissons un accès ouvert à l'information sur la façon dont l'argent des contribuables est dépensé afin que vous puissiez mieux tenir le Parlement, le gouvernement et les fonctionnaires responsables.
